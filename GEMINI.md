# Project Context: Convex Starter Skeleton

This project is a clean starter skeleton for building modern web applications.

## Tech Stack

- **Framework**: React 19 (Vite)
- **Backend**: Convex
- **Auth**: Convex Auth (@convex-dev/auth)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Toast**: Sonner

## Development Patterns

- **Convex Auth**: Use `Authenticated` and `Unauthenticated` components for conditional rendering.
- **Data Fetching**: Use `useQuery` and `useMutation` hooks from `convex/react`.
- **TypeScript**: Strictly typed backend and frontend.
- **Surgical Updates**: When modifying, stick to existing patterns.

## Starter Content

- A simple "Tasks" feature to demonstrate Convex queries and mutations.
- Basic Password authentication flow.
- Clean Tailwind layout.
